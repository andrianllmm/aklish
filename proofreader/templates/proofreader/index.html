How to Use Aklish Proofreader

  1. Start by typing or pasting your text into the provided text box.
  2. You can select the language of your text from the dropdown menu.
  3. You'll get instant feedback while you type:
    • Highlighted text means they are incorrect.
    • The number of words and mistakes.
    • The "correctness" % which is the ratio between the number of mistakes and words.
  4. Click the button to get suggestions.
  5. You can accept the proposed changes by clicking on them individually.

How it works

  • The spellcheckers used utilizes the Python library PySpellChecker.
  • The Aklanon spellchecker is based on the Aklish Aklanon dictionary.
    • It utilizes a stemmer algorithm where it derives the root word of an inflected word based on Aklanon inflection rules.
  • The English spellchecker is the default from PySpellChecker.
    • It utilizes a lemmatizer from the Python library LemmInflect where it derives the root word of an inflected word.

Considerations

  • Our proofreader is still in its early stages, and we're constantly working to improve its accuracy and functionality.
  • It may make mistakes or miss errors, so use it alongside your own judgment.
